*Kérlek, ha biztonsági rést találsz, jelezd nekem az Issues menün keresztül.*

# Reporting Security Vulnerabilities

# Please report any security vulnerabilities you find to me via the Issues menu.

Where you see ❗, that version is not recommended; where you see ❌, it is unsuitable for use—please choose a different version. If you see ⚠️, it is still a BETA version. If you see ✅, it is fully functional.
